# EduMatch

EduMatch — это современная платформа для поиска образовательных проектов, сборки команд и совместного управления задачами. Проект предоставляет инструменты для создания проектов, подачи заявок на участие, общения в реальном времени и управления задачами.

## 🚀 Основной функционал

- **Аутентификация и профили:** Регистрация, авторизация (JWT) и управление профилем пользователя (включая загрузку аватаров).
- **Управление проектами:** Создание, редактирование и удаление собственных проектов, а также поиск и просмотр доступных.
- **Система заявок:** Пользователи могут подавать заявки в проекты, а создатели проектов — одобрять или отклонять их.
- **Управление задачами (Kanban):** Создание задач и отслеживание их статуса с помощью удобного Drag-and-Drop интерфейса.
- **Real-time Чат:** Общение участников проекта в реальном времени с использованием WebSockets.
- **Система уведомлений:** Уведомления о новых заявках, изменениях статуса и других важных событиях.

## 🛠 Технологический стек

### Frontend
- **React 18** — библиотека для создания пользовательских интерфейсов.
- **Vite** — быстрый сборщик проектов.
- **Tailwind CSS** — utility-first CSS фреймворк для стилизации.
- **React Router** — маршрутизация.
- **React Query** — управление серверным состоянием и кэшированием.
- **React Hook Form** — работа с формами.
- **@hello-pangea/dnd** — реализация Drag-and-Drop (для доски задач).
- **Axios** — HTTP-клиент.

### Backend
- **Go (Golang)** — основной язык программирования бэкенда.
- **Gin** — высокопроизводительный веб-фреймворк для маршрутизации.
- **PostgreSQL** — основная реляционная база данных.
- **Redis** — in-memory хранилище (кэширование, сессии и т.д.).
- **WebSockets** — для реализации чата в реальном времени.
- **Swagger** — автогенерация документации к API (`/swagger/index.html`).

## ⚙️ Запуск проекта (Локально)

### Требования
Для локального запуска вам понадобятся:
- **Go** (версии 1.20+)
- **Node.js** (версии 18+) и npm/yarn
- **PostgreSQL** (убедитесь, что сервер БД запущен и создана нужная база данных)
- **Redis** (запущенный сервер)

### Шаги для запуска

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Ecodrugster/EduMatch.git
   cd EduMatch
   ```

2. **Настройка переменных окружения:**
   - Перейдите в папку `backend` и создайте/настройте файл `.env`, указав доступы к вашей БД PostgreSQL и Redis.
   - Аналогично проверьте наличие файлов `.env` в папке `frontend` (если требуются).

3. **Установка зависимостей фронтенда:**
   ```bash
   cd frontend
   npm install
   ```

4. **Быстрый запуск (для Windows):**
   В корне проекта находится файл `start.bat`. Просто запустите его:
   ```cmd
   start.bat
   ```
   Скрипт автоматически откроет два новых окна командной строки: одно для запуска Go-сервера, другое для запуска Vite dev-сервера фронтенда.

   *Альтернативный ручной запуск:*
   - **Backend:** `cd backend && go run cmd/api/main.go`
   - **Frontend:** `cd frontend && npm run dev`
